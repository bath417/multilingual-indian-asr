# 🎙️ Multilingual Indian ASR System

## 📌 Project Overview

This project implements a **Multilingual Indian Automatic Speech Recognition (ASR) System** that converts speech from Indian languages into text and translates the recognized text into multiple languages.

The system uses the **Whisper ASR model** to perform speech recognition and supports languages such as **Telugu and Tamil**. The recognized speech is then translated into multiple languages including **English, Hindi, Tamil, and Telugu**.

A simple **web interface built with Gradio** allows users to upload speech audio and view multilingual text outputs instantly.

---

## ✨ Features

- Multilingual speech recognition  
- Supports Indian languages such as **Telugu and Tamil**  
- Automatic **speech-to-text conversion**  
- **Multilingual translation** of recognized text  
- Interactive **web interface using Gradio**
